# 工作空間切換


## 我個人定義的個工作空間

| 工作空間 | 名稱  |
| -------- | ----- |
| 1        | File  |
| 2        | Edit  |
| 3        | Web   |
| 4        | Term  |
| 5        | Misc  |


## 循環切換

* [設定片段](https://github.com/samwhelp/note-about-hyprland/blob/gh-pages/_demo/config/hyprland-config/main/hyprland.conf#L191-L196)


| 按鍵組合  | 功能                 | 執行指令                   |
| --------- | -------------------- | -------------------------- |
| `Alt + a` | 切換到上一個工作空間 | `GoToDesktop` `left` (hyprland 內建) |
| `Alt + s` | 切換到下一個工作空間 | `GoToDesktop` `right` (hyprland 內建) |


> 也可以在「桌面」，使用「滑鼠中鍵」，上下滾動，切換「工作空間」。
